Task:   Editing an Object in IWM.

Description:   Allows users to edit exiting Objects in the system.

1.Filter the hierarchy to find the Object desire from the "Objects" screen.  
 
2.Select the "Edit" link on the row of the object you with to edit.      

This will display the "Edit Object" screen.      
 
 
 
3.Modify the required fields or the location of the object using the Locator drop-down menus.  
 

·"Name" is the identification that will be displaced on the field when scheduling work for this Object.  

·"Type" identifies the type of object it is.  

·"Meter Reading" is the current life of the Object expressed in its proprietary life measurements (i.e. miles, hours, etc.)  

·"Active" indicates whether this object will be ready for implementation once creation is finished. Not selecting this check-box will allow you to save the object while you plan your operating procedures. This will also serve as an option to retire unused objects.  
   
4.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
