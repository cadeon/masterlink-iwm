Task:   Editing an existing Task Group for an Object Template in IWM.   

Description:   Allows users to edit and existing Task Group for a Template in the system.    


1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Template in which you wish to edit a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object Template).  
 
This will display the "Task Groups" screen.     

3.   Select the "Edit" link to the left of the Task Group "Description".

This will display the "Edit Task Group Template" screen.      
 
 

4.   Modify the "Task Group Name" field if desired (required field).

5.   Use the boxes to select, or deselect, the Task you wish to include or remove from the group.    

6.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
