Task:   Assigning a skill and skill level to a worker in IWM.
   
Description:   Allows users to add, edit or remove the skills and level of skills of a worker in the system. This allows a worker to become a candidate for work requiring a particular skill and skill level.    

1.Select the Worker.  
 
   use the filters to shorten your search for the worker who's skills you want to assign.     

2.   Select the "Wrench" (wrench is facing the wrong way)button.

From the "Worker" screen locate and select the "Wrench" button under the "Skills" column for the worker you wish assign skills to.  
 
This will display the "Edit Skills" screen with the list of skills that person has already been assigned if any.      

3.   Select "trash can" from the right side of the skill you are wanting to Delete.
   

 

4.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
