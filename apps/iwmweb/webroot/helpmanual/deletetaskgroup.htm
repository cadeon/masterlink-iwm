Task:   Deleting an existing Task Group from an Object in IWM.   

Description:   Allows users to delete an existing Task Group from Objects in the system.
   
1.Filter the hierarchy to find the object you desire from the "Objects List" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Object in which you wish to delete a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object.  
 
This will display the "Object Task Group" screen.  
 
     

   
3.   Select the "delete" button .
