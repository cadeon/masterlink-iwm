Task:   Editing an existing Template Data from an Template in IWM.   

Description:   Allows users to edit existing Templates Data from a Template in the system.    

1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Data link "" under the "Data" column for the Template in which you wish to edit a Template Data.  
 
Usually expressed as a number representing the number of Template Data for that Template.  
 
This will display the "Template Data" screen  
 
.     

3.   Select the "Edit" link for the Template Data you wish to edit.
   
This will display the "Edit Template Data" panel.  
 
 
 
4.Modify the information as required in the form (all fields are required).     

5.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
