Task:   Deleting an existing Action from a Task Template in IWM.
   
Description:   Allows users to delete existing Actions from a Task Template in the system.
   
1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to delete an action.  
 
Usually expressed as a number representing the number of Tasks for that Object Template).  
 
This will display the "Template Tasks" screen.  
 
 

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to delete an action.

   Usually expressed as a number representing the number of Actions for that Object Template.

This will display the "Action Templates" screen.  
 
 

4.   Select the "delete" link on the rightmost column.

5. Select the "Ok" button to record your changes, or select the "cancel" button to discard your changes.

Verify that the number under the "Instances" columns is zero (0). If it is not zero it the system disables the deletion capability to safeguard the system from errors.
