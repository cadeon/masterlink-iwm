Task:   Editing an existing Task Group from an Object in IWM.   

Description:   Allows users to edit and existing Task Group from an Object in the system.
   
1.Filter the hierarchy to find the object you desire from the "Object List" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Object in which you wish to edit a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object.  
 
This will display the "Object Task Group" screen.  
 
 

3.   Select the "Edit" link to the left of the Task Group "Description".

This will display the "Edit Task Group" screen.  
 
     

4.   Modify the "Description" field if desired (required field).

5.   Use the Check boxes on the left to select from the Tasks available that you wish to include or remove from the group.

6.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
