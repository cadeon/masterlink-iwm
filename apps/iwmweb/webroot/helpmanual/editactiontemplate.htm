Task:   Editing an existing Action for a Task Template in IWM.   

Description:   Allows users to edit existing Actions for a Task Template in the system.    


1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to edit an action.  
 
Usually expressed as a number representing the number of Tasks for that Object Template).  
 
This will display the "Task Templates" screen.     

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to edit an action.

   Usually expressed as a number representing the number of Actions for that Object Template).

This will display the "Action Templates" screen.     


4.   Select the "Edit" link on the right side of the name.

This will display the "Edit Action" panel.  
 
 
 
5.Modify the information required from the form (all fields are required).  
 
6.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
