Task:   Deleting an existing Object Data from an Object in IWM.   

Description:   Allows users to delete existing Object Data from an Object in the system.    

   
1.Filter the hierarchy to find the object you desire from the "Object List" screen.  
 
2.Select the Data link "" under the "Data" column for the Object in which you wish to delete a Object Data.  
 
Usually expressed as a number representing the number of Object Data for that Object.  
 
This will display the "Object Data" screen.  
 
     

3.   Select the "delete" link for the Object Data you wish to delete.

This will delete the Object Data.
