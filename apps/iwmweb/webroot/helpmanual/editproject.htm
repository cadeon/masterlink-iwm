Task:   Editing a Project in IWM.   

Description:   Allows users to Edit Projects in the system.    
   
1.From the "Project List" screen locate the Project you wish to delete.      
 
2.Select the "Edit" link located on the rightmost column.  

This will display the "Edit" screen.  
 
     

2.Modify the fields required.  

·"Name" indicates the edification for this project.  

·"Description" User description of the project being created.  

·"Project Type" indicates the category of project it will be.  

·"Created Date" indicates the date in which the Project was created in the system. It is defaulted to today.  
 
·"Created By" is a system generated field which identifies who created the present Project.  



3.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
