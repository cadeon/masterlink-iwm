Task:   Editing a worker in IWM.   

Description:   Allows users to edit the information for a worker in the system.    

1.Select the Define - worker.  

From the "Worker" screen shorten the list by using the organization filters in which the person you wish to edit is located.  

2.   Select the "Edit" link located on the 2nd from far right of the person's name.

      This will display the "Edit Person" screen.
      
3.   Edit the information fields you wish to change

 
 
4.Select the "save" button to record your changes, or select the "cancel" button to discard your changes. .
