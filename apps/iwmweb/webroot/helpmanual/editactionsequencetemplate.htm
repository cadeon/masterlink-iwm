Task:   Editing the Order of Actions for a Task Template in IWM.
   
Description:   Allows users to edit the order of Actions for a Task Template in the system.    


1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to edit the sequence of actions.  
 
Usually expressed as a number representing the number of Tasks for that Object Template.  
 
This will display the "Template Tasks" screen.  
 
     

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to edit the sequence of actions.

   Usually expressed as a number representing the number of Actions for that Object Template.

This will display the "Action Templates" screen.  
 
 


4.   Select the "Order Actions" button .
 
This will display the "Reorder Actions" columns and control buttons on the list of Actions.  
 
5.Modify the Order using the drop down box for the Action you wish to move.  
 

 

6.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
