Task:   Editing the Service Calendar for an existing Template in IWM.
   
Description:   Allows users to edit the Service Calendar for an exiting Template in the system.
      
1.From the "Template List" screen filter the hierarchy to find the template you desire.  
 
2.Locate the Template you wish to edit and select the "Clndr" link located to the right of the Template name.  
 
This will display the "Service Calendar" screen.  
 
 

3.   Modify the information required from the form to indicate in which period of time this template should be active.
 
 
4.Select the "Save" button to record your changes or the "Cancel" button to discard them.
