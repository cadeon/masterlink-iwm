Task:   Deleting an existing Task from an Object in IWM.   

Description:   Allows users to delete new Task from Objects in the system.


1.Filter the hierarchy to find the Object you desire.  
 
2.Select the Task link "" under the "Task" column for the Object in which you wish to delete a task  
 
Usually expressed as a number representing the number of Tasks for that Object.  
 
This will display the "Object Task" screen.  
 
 
 
3.Select the "delete" link on the rightmost column for the Task you wish to delete.  
 
Verify that the number under the "Actions", columns is zero (0). If it is not zero the system disables the deletion capability to safeguard the system from errors.  
 
Note: If the task has been used to schedule work in the past, then it CANNOT be deleted, and may only be deactivated.
