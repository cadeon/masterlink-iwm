Task:   Editing a existing Object Data from an Object in IWM.   

Description:   Allows users to edit existing Object Data from an Object in the system.    

1.Filter the hierarchy to find the object you desire from the "Objects" screen.  
 
2.Select the Data link "" under the "Data" column for the Object in which you wish to create a Object Data.  
 
Usually expressed as a number representing the number of Obejct Date for that Object.  
 
This will display the "Object Data" screen.  
 
 

3.Select the "Edit" link for the Object Data you wish to edit.  
   
This will display the "Edit Object Data" panel.  
 
 
 
4.Modify the information as required in the form (all fields are required).     
 
5.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
