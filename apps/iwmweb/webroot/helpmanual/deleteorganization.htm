Task:   Deleting an existing Organization in IWM.    

Description:   Allows users to delete an exiting organization from the system. Some requirements may need to be met for a deletion to be allowed. Newly created organizations that have no employees and have not been used by the system for management may be readily deleted. If employees are recorded under that organization, they must be removed either by deleting them from the system or assigning them to another organization.    


1.   Select the "Organizations" link within the Define Section of IWM.

   The system will display the "Organization List" screen.    

2.   Locate the Organization you wish to delete.

3.   Select the "delete" link on the right side of the organization you wish to delete.

If the organization is ready for deletion it will be eliminated from the list. If other requirements need to be met for deletion, a warning message will display indicating what is required.
