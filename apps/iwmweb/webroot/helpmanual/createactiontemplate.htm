Task:   Adding a new Action for a Task Template in IWM.   

Description:   Allows users to add new Actions for a Task Template in the system.    

1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to create an action.  
 
Usually expressed as a number representing the number of Tasks for that Object Template.  
 
This will display the "Template Task" screen.     
 
 

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to create an action.

   Usually expressed as a number representing the number of Actions for that Object Template.

This will display the "Template Task Actions" screen.  
 
     
      
 
4.Select the "Add Action" button .  

This will display the "Add Action" panel.      
 
 

5.   Complete the information required from the form (all fields are required).

6.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
