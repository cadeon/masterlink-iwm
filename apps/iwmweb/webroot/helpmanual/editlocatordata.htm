Task:   Editing an existing Locator Data in IWM.   

Description:   Allows users to edit an exiting Locator Data for a locator in the system.
   
1.Filter the hierarchy of locators to find the locator you desire in the "Locators" screen.  
 
2.Select the Locator Data button "" from the "Data" column  

Usually expressed as three lines with a number next to it to represent the number of Locator Data for that locator.  
 
This will display the "Locator Data" screen.  
 
     
 
3.Select the "Edit" button on the right column for the Locator Data you wish to edit.  

This will display the "Edit Locator Data" panel.  
 
     

2.   Modify the information required from the form (all fields are required).    
 
 
3.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
