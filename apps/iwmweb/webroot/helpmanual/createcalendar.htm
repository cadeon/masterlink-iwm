Task:   Creating work days for a worker's Calendar in IWM.
   
Description:   Allows users to create working availability of a worker for a particular day (or days) and location in the system.    


1. Select the Worker.
      
From the "Worker" screen select the calendar button from the "Clndr" column for the worker you wish to create a calendar for.  

2.Select the "Add calendar" button(put pic here).  
 
3.Select a location for that person's calendar.     
 
4.Select a "Shift", the duration of the shift in "Time", and a range of days from the "Start Date" and "End Date" drop down menus      
 
5.Select the "Save" button to complete the transaction and "cancel" button to discard them.
