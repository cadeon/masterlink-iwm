Task:   Deleting Locator Data in IWM.
   
Description:   Allows users to delete existing Locator Data for a locator in the system.

1.Filter the hierarchy of locators to find the locator you desire in the "Locators" screen.  
 
2.Select the Locator Data button "" from the "Data" column  

Usually expressed as three lines with a number next to i to represent the number of Locator Data for that locator).  
 
This will display the "Locator Data" screen.  
 
3.Select the "delete" link on the column for the Locator Data you wish to delete.  
 
 
4Select the "Ok" button to confirm, or select the "cancel" button to cancel the request.
