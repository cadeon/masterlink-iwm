Task:   Editing an existing Task from an Object Template in IWM.

Description:   Allows users to edit existing Tasks from a Template in the system.

1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task button "" under the "Task" column for the Template in which you wish to edit a task  
 
Usually expressed as a number representing the number of Tasks for that Object Template).  
 
This will display the "Template Task" screen.     
 
3.Select the "Edit" link for the Task you wish to edit located to the right of the Task Description.  

This will display the "Edit Task" screen.  
 
 

4.   Modify the information required in the form.
 
·"Description" Is required and is what will be displayed to the users in the field.  
 
·   "Est. Time" is a required field and indicates the estimated duration a task should take to complete. The system uses this information to estimate the amount of workload given to a person.  
 
·"Skill" is a required field and indicates the type of skill required for that task.  
 
·"Skill Level" is a required field and much like the type of skill the level of that skill is indicated to specify a level of proficiency needed for that task.  
 
·"Worker Type" is a required field and indicates the type of organizational association a worker must have to work on that task.  
 
·"Number of Workers" is a required field and indicates the number of persons required for that task.  
 
·"Priority" is a required field and assigned an urgency value necessary for the system's schedule automation component.  
 
·"Task Type" categorizes what type of task it is. "Routine Tasks" have special frequency considerations that make the "Frequency" fields required.  
 
·"Frequency" is a required field ONLY IF the "Task Type" is "Routine Maintenance".  
 
There are two options for choosing a frequency.  
1- Select from the pre-selected frequencies in the Frequency drop-down menu.  
2- Specify an specific frequency interval units use (Meter Interval).      

5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
