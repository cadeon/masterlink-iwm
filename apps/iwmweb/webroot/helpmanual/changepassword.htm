Task:   Changing the users login password.
   
Description:   Allows users to change their passwords as needed protect the system from unauthorized use.    
1.Select the "Change Password" link on the upper right corner of the screen.  
 
 
 
The system will display a new window with the "change password" panel.  
 

2. Enter your current password into the "current password" text field.

3.   Enter a new password in the "New Password" text field.   

4.   Renter the same password in the "Retype New" text field.    

5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
