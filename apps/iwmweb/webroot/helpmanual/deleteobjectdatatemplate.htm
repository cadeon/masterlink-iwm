Task:   Deleting an existing Template Data from an Template in IWM.   

Description:   Allows users to delete existing Data from a Template in the system.
   
1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Data link "" under the "Data" column for the Template in which you wish to delete a Data Template.  
 
Usually expressed as a number representing the number of Template Data for that Template.  
 
This will display the "Template Data" screen.     
 
 

3.   Select the "delete" link for the Template Data you wish to delete.

4. Select the "Ok" button to record your changes, or select the "cancel" button to discard your changes.

   This will delete the Template Data.
