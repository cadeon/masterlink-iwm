Task:   Creating a Maintenance Request work order in IWM.
   
Description:   Allows users to create a unscheduled work as a Maintenance Request on IWM.    

1.Select the "Submit New Request" button .  

This will display the "Create Maintenance Request" Panel.  
 
 

2.   Complete the information for the required fields.

·"Email" indicates the contact email from the customer initiating the request.  
 
·"Name" indicates the name of the customer initiating the request.  
 
·"Phone" indicates the phone number of the customer initiating the request.  
 
·"Problem" indicates the type of problem using a predefined category which routes the request to the most qualified party.  
 
·"Comment" necessary field for brief description of details of the request.  
 
·"Request Location" indicates using the Location Hierarchy where the request is necessary.      
 
3.Select the "Save" button to record your changes, or select the "Reset" button to discard your changes.
