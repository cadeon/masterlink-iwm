Task:   Editing an existing Action for a Task under an Object in IWM.
   
Description:   Allows users to edit existing Actions for a Task under Objects in the system.   

Note: Actions may only be created, deleted or edited for "Custom" and "Stand Alone" tasks.    

1.Filter the hierarchy to find the object you desire from the "Objects List" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to edit an action.  
 
Usually expressed as a number representing the number of Tasks for that Object.  
 
This will display the "Object Task" screen.  
 
     

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to edit an action.

   Usually expressed as a number representing the number of Actions for that Object.

This will display the "Object Task Actions" screen.  
 
     


4.   Select the "Edit" link on the row of the action you wish to edit.

This will display the "Edit Action Data" panel.  
 
 
 
5.Modify the information required from the form (all fields are required).  
 
6.Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
