Task: Remove a Project Stencil in IWM

Description: Allows users to remove a task from a Project Stencil in the system.

1.Select the "Task" button in the Project Stencils screen for the Project Stencil you wish to edit.  
 
This will display the "Stencil Tasks" screen.  
 
 

2.   Select the "remove" button for the task you wish to remove.

3.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
