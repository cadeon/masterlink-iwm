Task:   Creating an Object in IWM.   

Description:   Allows users to create new Objects in the system.    
   
1.   Select the "Add Object" button from the "Object" screen.
 
This will start the "Create Object Sequence". Starting with the:  
 
"Create Object: Pick Template" screen.      
 
 

2.   Filter the hierarchy to find the Object Template that corresponds to the type of Object you wish to create.    

3.   Select the Template using the "circle" to the left of the desired Template.

4.    Select the "Next" button.

This will display the:  
"Create Object: Set Properties" screen.      
 
 
   
4.Complete the required fields and Identify the location of the object using the Locator drop-down menus.  

·"Name" is the identification that will be displaced on the field when scheduling work for this Object.  
 
·"Type" identifies the type of object it is.  
 
·"Meter Reading" is the current life of the Object expressed in its proprietary life measurements (i.e. miles, hours, etc.)  
 
·"Active" indicates whether this object will be ready for implementation once creation is finished. Not selecting this check-box will allow you to save the object while you plan your operating procedures.      
 
5.Select the "Finsh" button to record your changes, or select the "cancel" button to discard your changes.  

This will create a new Object which would be an instance of the Object Template you selected. It will have the same properties as defined in the Template including Tasks and Actions.
