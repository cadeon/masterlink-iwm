Task:   Sorting a list of items in IWM
   
Description:   Allows the sorting of listed items in IWM.   

1.From a list of items locate the column you would like to use for sorting.  
 
2.Select the column title link on the top of the column.  
 
The list of items will be sorted in the opposite order by the values on that column.  
Hint: Only one column may be used for sorting.
