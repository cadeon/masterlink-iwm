Task:   Deleting an existing Task Group for an Object Template in IWM.   

Description:   Allows users to delete and existing Task Group for a Template in the system.    
   
1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Template in which you wish to delete a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object Template.  
 
This will display the "Task Groups" screen.     

3.   Select the "delete" button .

Verify that the number under the "Instances" columns is zero (0). If it is not zero the system disables the deletion capability to safeguard the system from errors.
