Task:   Adding a new Locator Data in IWM.
   
Description:   Allows users to add new Locator Data for a locator in the system.    

1.Filter the hierarchy of locators to find the locator you desire in the "Locators" screen.  
 
2.Select the Locator Data buttonfrom the "Data" column  

Usually expressed as three lines with a number next to it to represent the number of Locator Data for that locator).  
 
This will display the "Locator Data" screen.  
 
 
 
3.Select the "Add Locator Data" button .  

This will display the "Add Locator Data" panel.      

4.   Complete the information required from the form (all fields are required).    

 

5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
