Task: Creating a New Project Stencil in IWM

Description: Allows users to create new Project Stencil in the system.

1.Select the "Add Project Stencil" button in the Project Stencil screen.  
 
This will display the Add Project Stencil Panel  
 
 

2.   Complete the information required from the form (all fields are required).
   

3.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
