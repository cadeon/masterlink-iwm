Task: Add a New Task to a Project Stencil in IWM

Description: Allows users to add a new task to a Project Stencil in the system.

1.Select the "Task" button in the Project Stencils screen for the Project Stencil you wish to edit.  
 
This will display the Edit Task panel.  
 
 
 

2.   Complete the information required from the form (all fields are required).

3.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
