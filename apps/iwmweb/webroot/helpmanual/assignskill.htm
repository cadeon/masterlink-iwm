Task:   Assigning a skill and skill level to a worker in IWM.
   
Description:   Allows users to add, edit or remove the skills and level of skills of a person in the system. This allows a worker to become a candidate for work requiring a particular skill and skill level.    

1.Select the Worker.  
 
   use the filters to shorten your search for the worker who's skills you want to assign.     

2.   Select the "Wrench" (wrench now points other way) button

From the "worker" screen locate and select the "Wrench" button under the "Skills" column for the person you wish assign skills to.  
 
This will display the "Edit Skills" screen with the list of skills that person has already been assigned if any.      
 
3. Select the "Drop Down Menu" .  
 
The list of available skills will be displayed.  
 
4.Click on the skill you wish to assign to the person.  
 
The list of available skills will disappear and the list of assigned skills will be displayed including the newly selected skills. By default the skill level will be set to one.  
 
5.Select the "save" button to record your changes, or select the "cancel" button to discard your changes. .
