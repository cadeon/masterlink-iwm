Task:   Deleting an existing Object in IWM.
   
Description:   Allows users to delete new Objects in the system.
   

1.Filter the hierarchy to find the Object you desire.      
 
2.Select the "delete" link on the rightmost column for the object you wish to delete.  
      
 
Verify that the number under the "Tasks", columns is zero (0). If it is not zero the system disables the deletion capability to safeguard the system from errors.  
 
Note: If the object has been used to schedule work in the past, then it CANNOT be deleted, and may only be deactivated.
