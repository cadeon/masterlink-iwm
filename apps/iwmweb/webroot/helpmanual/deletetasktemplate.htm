Task:   Deleting an existing Task from an Object Template in IWM.   

Description:   Allows users to delete existing Tasks from a Template in the system.    

1.Filter the hierarchy to find the template you desire from the "Template List" screen.  
 
2.Select the Task link "" under the "Task" column for the Template in which you wish to delete a task  
 
Usually expressed as a number representing the number of Tasks for that Object Template.  
 
This will display the "Templates Tasks" screen.  
 
 
 
3.Select the "delete" link on the rightmost column to delete the task.  
 
4.   Select the "Ok" button to record your changes, or select the "cancel" button to discard your changes.  
 
Verify that the number under the "Actions" and "Instances" columns is zero (0). If it is not zero the system disables the deletion capability to safeguard the system from errors.
