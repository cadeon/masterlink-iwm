Task:   Adding Task Group for an Object Template in IWM.
   
Description:   Allows users to Add a new Task Groups for a Template in the system.    

1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Template in which you wish to create a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object Template.  
 
This will display the "Task Groups" screen.  
 
     

3.   Select the "Add Task Group" button .

This will display the "Add Task Group Template" panel.  
 
 

4.   Enter a "Task Group Name" for the group on the field provided. (required field).      
   
Using the boxes located in the first column, mark the Tasks you wish to include in the group.  
 
5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
