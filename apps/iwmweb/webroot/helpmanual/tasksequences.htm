Project Stencils
Previous  Top  Next

Description:   

Project Stencils are a subsection designed to create repeatable sequences of tasks that may incorporate different objects and resources. Each Project Stencil is labeled and contains within it a list of all the tasks that must be completed for that sequence. In addition it includes a "step" mechanism that allows one to group tasks in chronological clusters. This capability enables the system to enforce the completion of a single or group of tasks, before any other of the subsequent tasks are scheduled.
