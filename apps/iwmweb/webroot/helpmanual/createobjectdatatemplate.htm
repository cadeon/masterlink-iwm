Task:   Adding new Template Data for an Template in IWM.   

Description:   Allows users to add new Data for a Template in the system.    

1.Filter the hierarchy to find the template you desire from the "Templates" screen.  
 
2.Select the Data link "" under the "Data" column for the Template in which you wish to create a Data Template.  
 
Usually expressed as a number representing the number of Data Template for that Object Template.  
 
This will display the "Template Data" screen.  
 
     

3.   Select the "Add Template Data" button .

This will display the "Add Data Template" panel.  
 
 

4.   Complete the information required from the form (all fields are required).

5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
