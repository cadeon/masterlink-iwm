Task:   Creating a Task Group for an Object in IWM.
   
Description:   Allows users to create new Task Groups for an Object in the system.    

1.Filter the hierarchy to find the object you desire from the "Object List" screen.  
 
2.Select the Task Group link "" under the "Task Groups" column for the Object in which you wish to create a Task Group.  
 
Usually expressed as a number representing the number of Task Groups for that Object.  
 
This will display the "Object Task Groups" screen.  
 
     

3.   Select the "Add Task Group" button .

This will display the "Add Task Group" panel.  
 
 

4.   Complete the information required from the form (all fields are required).

Enter a "Task Group Name" for the group on the field provided. (required field).      
   
Using the check boxes on the left, select from the Tasks available that you wish to include in the group.  
 

5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
