Task:   Creating a new Task for an Object in IWM.
   
Description:   Allows users to create new Tasks for an Object in the system.

Note: This process will make the object a "Custom Object" (Marked by an "*" next to the Task link) since it will disable the link to the Object Template. Thus any modification to this object's template will not be inherited by this object.

1.Filter the hierarchy to find the object you desire from the "Objects" screen.  
 
2.Select the Task link "" under the "Task" column for the Object in which you wish to create a task  
 
Usually expressed as a number representing the number of Tasks for that Object.  
 
This will display the "Object Task" screen.  
 
 
 
3.Select the "Add Task" button .  

This will display the "Add Task" screen.      
 
 

4.   Complete the information required in the form.

·"Description" Is required and is what will be displayed to the users in the field.  
 
·"Task Type" categorizes what type of task it is. "Routine Tasks" have special frequency considerations that make the "Frequency" fields required.  
 
·"Priority" is a required field and assigned an urgency value necessary for the system's schedule automation component.  
 
·   "Est. Time" is a required field and indicates the estimated duration a task should take to complete. The system uses this information to estimate the amount of workload given to a person.  
 
·"Num. of Workers" is a required field and indicates the number of persons required for that task.  
 
·"Skill" is a required field and indicates the type of skill required for that task.  
 
·"Skill Level" is a required field and much like the type of skill the level of that skill is indicated to specify a level of proficiency needed for that task.  
 
·"Worker Type" is a required field and indicates the type of organizational association a worker must have to work on that task.  
 
·"Frequency" is a required field ONLY IF the "Task Type" is "Routine Maintenance".  
 
There are two options for choosing a frequency.  
1- Select from the preformatted frequencies in the Frequency drop-down menu.  
2- Specify a specific frequency interval using the Service Interval Fields either by time (Months Between Service, and Days Between Service) or use (Meter Reading Increment).      

·"Schedule By" is a required field and indicates who is responsible for scheduling the task once it has been created as a Job. "Manual" scheduling gives the most control but does not allows the system to perform to peak performance when optimizing schedules. "System" allows the system to automatically handle scheduling through its optimization processes.  
 
·"Last Planned Date" is an automatically generated field by the system, and indicates the last time this task was planned (i.e. a Job was created using this task).  
 
·"Service Plan" is a required field and indicates the timing constraints regulating this particular task (i.e. when this task should be active).  


5.   Select the "save" button to record your changes, or select the "cancel" button to discard your changes.
