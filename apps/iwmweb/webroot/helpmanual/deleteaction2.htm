Task:   Deleting an existing Action from a Task under an Object in IWM.   

Description:   Allows users to delete existing Actions for Tasks under Objects in the system.

1.Filter the hierarchy to find the object you desire from the "Objects" screen.  
 
2.Select the Task link "" under the "Task" column for the Object in which you wish to delete an action.  
 
Usually expressed as a number representing the number of Tasks for that Object.  
 
This will display the "Object Task" screen.  
 
     

3.   Select the Actions link "" under the "Actions" column for the Task in which you wish to delete an action.

   Usually expressed as a number representing the number of Actions for that Object.

This will display the "Object Task Actions" screen.  
 
 
 

4.   Select the "delete" link on the rightmost column.


Actions may only be created, deleted or edited for "Custom" and "Stand Alone" tasks. An error meessage will display to indicate this.
