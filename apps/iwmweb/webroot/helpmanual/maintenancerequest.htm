Maintenance Request
Previous  Top  Next

Description:   

Maintenance Request is the sub-section where unscheduled and undefined work orders are entered in the system. This section is used for work requests initiated by an end customers where the manager is unsure of the exact nature of the request or the exact component in the system that is involved (thus, no job can be created from an object in the system). The Maintenance Request sub-section contains a categorical menu of the major types of issues that may arise in an environment (This are defined by the user and are specific to the environment in which IWM is used).
