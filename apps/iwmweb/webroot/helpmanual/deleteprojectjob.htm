Task:   Deleting an existing Job from a Project in IWM.
   
Description:   Allows users to delete existing Jobs from Projects in the system.    
   
1.From the "Project List" screen locate the Project in which you wish to delete a job.      
 
2. Select the "Jobs" link "" under the "Jobs" column

Usually expressed as a number representing the number of Jobs for that Project.  

This will display the "Project Jobs" screen.  
 
 

3.   Filter the hierarchy to find the job you wish to delete in the "Project Jobs" screen.

4.   Select the "delete" link on the rightmost column.

Note: Jobs that have been completed or are in progress (i.e. a person has reported time or field notes) may not be deleted. Those jobs may only be "Expired" or "Canceled".
